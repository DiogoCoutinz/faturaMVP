# Equipa Francisco Brito - Gestão de Faturas

Sistema de gestão de faturas com integração Supabase e processamento inteligente.

## Tecnologias Utilizadas

- **Vite**
- **TypeScript**
- **React**
- **shadcn/ui**
- **Tailwind CSS**
- **Supabase**

## Como Começar

1. **Instalar Dependências**
   ```bash
   npm install
   ```

2. **Configurar Variáveis de Ambiente**
   Cria um ficheiro `.env` na raiz com:
   ```bash
   VITE_SUPABASE_URL=a_tua_url
   VITE_SUPABASE_ANON_KEY=a_tua_chave
   ```

3. **Executar em Desenvolvimento**
   ```bash
   npm run dev
   ```

4. **Build para Produção**
   ```bash
   npm run build
   ```
